# The delivery loop

<div class="feature-list" style="margin-top: calc(20 * var(--pt));">

<div class="feature">
  <div class="feature-num">01</div>
  <div>
    <div class="feature-title">Implement</div>
    <p class="feature-body">Make the smallest coherent change that satisfies the task and matches local conventions.</p>
  </div>
</div>

<div class="feature">
  <div class="feature-num">02</div>
  <div>
    <div class="feature-title">Test, lint, visually verify</div>
    <p class="feature-body">Run targeted tests first, broaden checks when the blast radius grows, and inspect rendered UI when pixels matter.</p>
  </div>
</div>

<div class="feature">
  <div class="feature-num">03</div>
  <div>
    <div class="feature-title">Review changes</div>
    <p class="feature-body">Inspect the diff, call out assumptions, and remove accidental churn before asking anyone else to review.</p>
  </div>
</div>

<div class="feature">
  <div class="feature-num">04</div>
  <div>
    <div class="feature-title">Commit or create PR</div>
    <p class="feature-body">When evidence is attached and the diff is explainable, turn the work into a durable handoff.</p>
  </div>
</div>

</div>
