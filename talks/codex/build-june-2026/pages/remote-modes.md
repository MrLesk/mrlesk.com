# Remote and branch modes

<div class="feature-list">

<div class="feature">
  <div class="feature-num">01</div>
  <div>
    <div class="feature-title">Cloud tasks</div>
    <p class="feature-body">Queue bounded repo work, let it run away from the laptop, then inspect the resulting patch like any other teammate change.</p>
  </div>
</div>

<div class="feature">
  <div class="feature-num">02</div>
  <div>
    <div class="feature-title">Forks</div>
    <p class="feature-body">Split an idea when the next move is uncertain: one branch can explore design, another can harden tests, another can stay clean.</p>
  </div>
</div>

<div class="feature">
  <div class="feature-num">03</div>
  <div>
    <div class="feature-title">Resume and reconcile</div>
    <p class="feature-body">Bring the winning thread back to the delivery loop: run checks, compare diffs, and decide whether it should become a commit or PR.</p>
  </div>
</div>

</div>
