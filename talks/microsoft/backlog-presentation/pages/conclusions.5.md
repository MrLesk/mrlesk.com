# Get your project AI-ready

<v-clicks>

* **Preparation**: Create `AGENTS.md` and add **skills** to teach the agent about your project workflows
* **Writing Code**: Enforce **code style** and allow runtime inspection via **Microsoft DebugMCP**
* **Verification**: Agents must run tests and **self-verify** to meet the "Definition of Done"
* **Feedback Loop**: Treat your prompts as code and iterate on them
* Recommended reading: [factory.ai/news/agent-readiness](https://factory.ai/news/agent-readiness)
